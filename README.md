"# Streamlit_GenAI_chatbot"
This is a chat bot generated using the Generative AI and the UI is built using Streamlit.
Using the Groq model api key.
Also the chatbot is deployed on the streamlit cloud platform.
Using the ConversationBufferWindowMemory from langchain.memory to save the history of the conversation.

You can visit this link to try: https://generative-ai-chatbot-ojnkwdezg6lg5edjkwartt.streamlit.app/
